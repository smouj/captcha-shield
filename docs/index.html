7 Defense Layers

Each layer independently blocks attacks — a single bypass is never enough

Layer 1

Headless Detection

40+ detection vectors for headless browsers, automation frameworks, and bot fingerprints.

Layer 2

Behavioral Pre-check

28 behavioral signals analyzed via Bayesian scoring before any challenge is shown.

Layer 3

Dynamic Challenge Selection

Challenge type and difficulty dynamically chosen based on real-time risk assessment.

Layer 4

Interactive Challenge

One of 10 AI-proof challenges presented with behavioral verification during interaction.

Layer 5

QR Mobile Verification

High-risk sessions require cross-device verification via encrypted QR code.

Layer 6

WebAuthn / Passkey

Hardware-backed authentication as an optional additional verification layer.

Layer 7

Cryptographic Token

JWT HMAC-SHA256 token issued with session fingerprint and risk score embedded.
