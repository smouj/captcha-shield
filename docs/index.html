Cómo funciona

1

Fingerprinting de dispositivo

Se analiza WebGL, Canvas, plugins, hardware concurrency, resolución, timezone y se detectan navegadores headless y herramientas de automatización.

2

Generación de desafío

Se genera aleatoriamente uno de 7 tipos de desafío interactivo o se activa la verificación QR con código de 6 dígitos.

3

Monitoreo de 14 señales

Durante la interacción se capturan movimientos, clics, teclado, presión, scroll, visibilidad de pestaña y más datos comportamentales.

4

Scoring compuesto

Las 14 señales se combinan con pesos específicos para generar una puntuación de riesgo con 4 niveles: bajo, medio, alto y crítico.

5

Verificación final

Se combina el resultado del desafío con el análisis comportamental. Solo las interacciones que pasan ambos filtros son verificadas.

Fórmula de scoring (14 señales)

riesgo = (

linealidad x 0.10 + timing x 0.10 + velocidad x 0.08 +

hesitación x 0.12 + entropía x 0.06 + bézier x 0.06 +

dispositivo x 0.08 + teclado x 0.08 + puntero x 0.06 +

scroll x 0.04 + presión x 0.04 + pestaña x 0.06 +

entorno x 0.06 + temporal x 0.06

)

Niveles: bajo (<30%) · medio (30-50%) · alto (50-70%) · crítico (>70%)
