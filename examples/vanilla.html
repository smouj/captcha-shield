🛡️ CAPTCHA Shield - Vanilla HTML Example

Important: Client-Side Only

This is a demo/example showing how to embed CAPTCHA Shield in a vanilla HTML page. The widget runs entirely client-side and is NOT suitable for production security. For production, you need server-side verification.

Quick Embed

<!-- 1. Add CAPTCHA container --> <div id="captcha-shield"></div> <!-- 2. Load widget script --> <script src="https://smouj.github.io/captcha-shield/widget.js"></script> <!-- 3. Handle verification result --> <script> window.onCaptchaVerified = function(response) { // response = { success, riskScore, riskLevel, token } if (response.success) { // User passed CAPTCHA console.log('CAPTCHA passed!', response); } else { // User failed or aborted console.log('CAPTCHA failed or aborted', response); } }; </script>

Complete Example

<!DOCTYPE html> <html> <head> <title>My Form</title> </head> <body> <!-- CAPTCHA Container --> <div id="captcha-shield"></div> <!-- CAPTCHA Widget Script --> <script src="https://smouj.github.io/captcha-shield/widget.js"></script> <!-- Verify Callback --> <script> window.onCaptchaVerified = function(response) { if (response.success) { // Submit form programmatically document.getElementById('myForm').submit(); } }; </script> <!-- Your Form (only submitted if CAPTCHA passes) --> <form id="myForm" action="/submit" method="POST"> <input type="text" name="username" placeholder="Username" required> <input type="email" name="email" placeholder="Email" required> <button type="submit">Submit</button> </form> </body> </html>

Advanced Configuration

<!-- Configuration BEFORE widget script --> <script> window.CaptchaShieldConfig = { theme: 'dark', // 'dark' | 'light' primaryColor: '#10b981', // Any hex color language: 'es', // 'es' | 'en' | 'pt' size: 'normal', // 'compact' | 'normal' | 'large' borderRadius: '16px', containerId: 'captcha-shield', timeout: 120000, // 2 minutes onVerify: function(response) { // Alternative to window.onCaptchaVerified console.log('Verified:', response); }, onError: function(error) { console.error('Error:', error); }, onLoad: function() { console.log('Widget loaded!'); } }; </script> <script src="https://smouj.github.io/captcha-shield/widget.js"></script>

Widget API

// Event listeners window.CaptchaShield.on('ready', function() { // Widget iframe loaded }); window.CaptchaShield.on('verified', function(response) { // User passed CAPTCHA }); window.CaptchaShield.on('failed', function(response) { // User failed CAPTCHA }); window.CaptchaShield.on('expired', function() { // Session expired }); window.CaptchaShield.on('resize', function(size) { // Widget resized }); // Programmatic reset window.CaptchaShield.reset();
